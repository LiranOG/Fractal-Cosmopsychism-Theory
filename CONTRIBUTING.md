# Contributing to the Fractal Cosmopsychism Theory Repository

Thank you for your interest in contributing to the FCT project. This document provides guidelines for maintaining the scientific rigor and epistemological discipline that define this repository.

## Core Principles

1. **The Epistemological Arrow** — Observation → Model → Formula. Never derive conclusions from axioms alone.
2. **Tier Classification** — Every claim must be assigned a Tier (A–D). Tier inflation is intellectually dishonest.
3. **Cold Truth Mandate** — Every document must include a "Cold Truth" section acknowledging what the model cannot do.
4. **X-Proxy Declaration** — Every measurement must declare its proxy status.

## File Naming Conventions

- SYN formulas: `SYN_XX_descriptive_name.md`
- Upgrade protocols: `UP_XX_descriptive_name.md`
- Paper protocols: `paper_protocol_XX_name.md`
- Integration bridges: `INTEGRATION_system_name.md`

## LaTeX Standards

- Use `$...$` for inline math, `$$...$$` for display math
- All variables must match the definitions in `variable_ontology.md`
- Dimensional analysis is mandatory for any new formula

## Pre-Registration

Any new empirical prediction must include:
1. Falsification criteria ($H_{\text{NULL}}$)
2. Required sample size with power analysis
3. Pre-registration plan (OSF or ClinicalTrials.gov)

## Pull Request Checklist

- [ ] Cold Truth section present
- [ ] Tier classification assigned
- [ ] Dimensional audit complete
- [ ] Cross-references to SYN_REGISTRY verified
- [ ] No Tier inflation (Tier D claims are not labeled Tier B)
