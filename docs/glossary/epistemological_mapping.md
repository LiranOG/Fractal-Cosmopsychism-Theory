## The Iron Rule — Before Everything Else

The only valid direction of scientific reasoning is:

```
Observed Phenomenon → Mathematical Model → Formula
```

**Not:**

```
Formula → [search for a phenomenon to fit it]
```

The reverse direction produces formally impressive pseudoscience. Every framework mapping in this document observes this rule: we identify the **phenomenon first**, then select the mathematical field that models it.
